🌙 Light Way — Black Screen Overlay for Your Browser
A simple, lightweight, and effective browser extension that darkens any web page with a fully adjustable black overlay.
Light Way creates a smooth, controllable black veil over your current tab, letting you dim the screen precisely — from a barely noticeable shadow to absolute darkness where the monitor appears completely off. No complicated settings, no ads, no data collection. Just you, a slider, and your comfort.

🤔 Why This Exists
The average person spends 8 to 14 hours a day staring at a screen. Monitor brightness, especially at night, causes eye strain, disrupts circadian rhythms, and degrades sleep quality. Built-in solutions — night mode, blue light filters, brightness sliders — help, but they don't solve the core problem. Sometimes you just want to "turn off" the screen for a moment without touching physical monitor buttons or closing applications.

Light Way was created exactly for this. It lets you:

Instantly darken any tab — when you need to look away, think for a moment, or rest your eyes without closing the page.
Create a comfortable brightness level — not every website has a dark theme. Global solutions like Dark Reader modify page structure. Light Way simply lays a transparent black layer on top without breaking anything.
Save battery on OLED screens — pixels displaying pure black are physically turned off on OLED panels. A fully black overlay genuinely reduces laptop power consumption.
Focus deeply on one task — a dimmed screen helps eliminate visual noise and sharpens mental focus.
Rest your eyes without pausing everything — audio keeps playing. Podcasts, music, calls, and videos continue while the screen stays completely black.
✨ Key Features
Feature	Description
🎚️ Smooth Adjustment	Slider from 0% to 100% with 1% increments — precise control over darkness depth.
⚡ Instant Response	Changes apply immediately as you drag the slider, with zero lag.
🧩 Non-Destructive	The overlay does not touch the website's DOM, styles, or layout. It simply sits on top.
🚫 0% = Fully Off	At zero, the overlay is completely removed from the page. Clicks pass through, and the website works as normal.
🌐 Works Everywhere	Chrome, Edge, Opera, Brave, Vivaldi, Yandex Browser, and any Chromium-based browser.
📴 Fully Offline	Zero network requests. No fonts, images, analytics, or external resources.
🎯 Single Purpose	The extension does exactly one thing and does it well. No feature bloat.
🖱️ Usage
Open any web page.
Click the Light Way icon in your browser toolbar.
Move the darkness slider in the popup:
Right — the page gets darker.
Left — the page gets lighter.
All the way left (0%) — the extension is fully off, and the overlay is gone.
Close the popup. The setting persists on that tab until you reload it.
Tips:

For maximum effect, use fullscreen mode (F11). The black overlay blends perfectly with your monitor bezels.
On OLED screens, 100% darkness is visually identical to a powered-off display.
Audio continues during blackout — perfect for podcasts, ambient music, or calls while resting your eyes.
🔐 Security and Privacy
Light Way is built on a strict principle of least privilege:

No data collection. The extension sends no network requests, uses no analytics, and stores no history.
No network permissions. The manifest grants zero access to the internet.
Runs only on your action. Code is injected exclusively when you move the slider.
No persistent storage. Darkness resets when the tab or browser is restarted.
Fully open source. Every file is available for inspection. No hidden functionality, no obfuscation.
⚠️ Limitations
Does not work on system pages. chrome://, chrome-extension://, and about: URLs are protected from script injection. This is a Chromium security restriction.
Does not cover the browser chrome. The tab bar, address bar, and bookmarks bar remain visible. Use F11 for a full-black experience.
Resets on tab reload. The darkness setting is lost when you refresh the page or navigate to a new URL (SPA navigation without URL changes works fine).
Chromium-only. Manifest is not supported in Firefox or Safari.
